OSIF

Con este directorio, podremos conseguir la información de nuestros amigos de facebook, aunque algunas veces Facebook lo detecta y te deshabilitan tu cuenta, pero te lo vuelven activar al momento si reconoces algunos de tus amigos, pero si no te importa eso e igual quieres hacerlo y tener los números y correos de tus amigos, siguenos con estos comandos.

$ pkg update && pkg udgrade -y
$ pkg install -y python2
$ pkg install -y git
$ git clone https://github.com/ciku370/OSIF.git
$ cd OSIF
$ pip2 install -r requirements.txt
$ python2 osif.py
$ token
$ Pongan su correo y contraseña hasta que les salga $
$ python2 osif.py
$ help (es para ver lo que pueden realizar)
$ dump_phone (Todos los números)
$ dump_mail (Todos los correos)
